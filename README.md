# Trình duyệt đơn giản sử dụng PyQt5 và Python

![screenshot](/assets/screenshot.png)

## Installation

Tạo `venv`

```bash
  python -m venv venv
```

Activate `venv`

```bash
  cd venv/Scripts && activate.bat
```

Cài các gói cần thiết `pip`

```bash
   pip install -r requirements.txt
```

Chạy

```bash
   python main.py
```
